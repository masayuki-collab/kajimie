    💡 ポイント = 時間（分）× 見えにくさ係数。 食事のように評価されやすい家事は係数1.0、洗濯物を畳む・トイレ掃除のような 「見えにくい家事」ほど係数が高く（最大2.0）、がんばりが数字に反映されます。

    ワンタップで記録

    👻 は「見えにくい家事」。左の✓を付けると記録され、外すと今日の分を取り消せます。複数回やったら ＋、リストは ✎ で編集・✕ で削除

    定番の家事を追加

    ワンタップのリストに追加します（名前・時間・見えにくさは ✎ からいつでも変更可能）

    👻 名もなき家事

    気づかれないけど誰かがやっている家事。タップで記録（全部 係数2.0）、✕ で削除できます

    自由に記録

    ユーザー設定

    夫婦それぞれの名前。画面右上で「いま入力する人」を切り替えられます

    データ管理

    初回起動時はイメージをつかむためのサンプルデータが入っています

    定期掃除の期限

    「風呂の防カビは1ヶ月に1回」「換気扇は3ヶ月に1回」のような定期掃除の期限を見える化。左の✓を付けると完了記録＆次の期限がセットされ、家事ポイントにもなります（👻 見えにくい家事）

    定期掃除を追加

    追加したタスクは「未記録」から始まり、はじめて ✓ を付けた日から周期のカウントが始まります（見えにくさ係数1.6で記録）。一覧の ✎ でいつでも編集できます

    日用品ストック

    使ったら − ボタンで記録。消費ペースを学習して、定期おトク便のおすすめ頻度を自動計算します。お届きボタンを押すと在庫が増え、補充作業も家事として自動記録されます（👻 係数2.0）
    在庫方針

    品目を追加

    価格やお届け量などの細かい項目は空欄でもOK。一覧の ✎ からいつでも編集できます
    ※「Amazonで注文」ボタンは Amazon.co.jp の検索ページを開く簡易連携です。 定期おトク便の頻度設定はこのアプリ内での目安管理で、実際の申込み・頻度変更・ スキップはAmazon側の定期おトク便設定で行ってください（おすすめ頻度をそのまま選べばOK）。 省スペース方針では、小さめのパック（例：トイレットペーパー4ロール）を短い間隔で 受け取るのがコツ。在庫が多めと出たら次回分をスキップすると棚が空きます。 本格的な連携（価格取得・ワンクリック注文・定期便の自動化）は Amazon Product Advertising API / Amazon Business API で拡張できます。

    家族に共有する

    レポートをテキストにして、家族のLINEグループなどに貼り付けられます
    データはこの端末のブラウザ内（localStorage）にのみ保存され、外部には送信されません。
    📲 スマホではブラウザのメニューから「ホーム画面に追加」でアプリとして使えます（iPhoneは共有ボタン→「ホーム画面に追加」）。一度開けばオフラインでも動作します。
